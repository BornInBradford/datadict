BiBBS

Container data dictionary: github | csv | xlsx

Various datasets collected for the Born in Bradford’s Better Start cohort study. The current released dataset is a datafreeze of the cohort study at 2019-11-30.

BiBBS_CohortInfo BiBBS Cohort Information Information about BiBBS cohort participants
