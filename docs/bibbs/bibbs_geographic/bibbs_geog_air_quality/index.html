BiBBS property level air quality information

Table ID: BiBBS_Geographic.bibbs_geog_air_quality

github | csv

Entity type Variables Entities Updated Related tables
Property 88 3478 2023-07-13 NA

Property level air quality data. Contains measures of mean and maximum exposures to PM10, PM2.5 and NOx, using Bradford Council (CBMDC) and Department for Environment (DEFRA) data. Variables cover a range from exposure radii (from 100m to 2000m) measured using both circular buffers and street networks centred on the home address.
Usage instructions: To link to BiBBS participants, use the id column to link to the property_id variable in bibbs_geog_person.
Further information: Created by Kimon Krenz, on behalf of Born in Bradford. Please reference Krenz et al. (2023) when using this data, where more detailed methodological information can be found regarding the construction of these variables.
