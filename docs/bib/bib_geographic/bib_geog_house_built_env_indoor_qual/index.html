BiB house level built environment and indoor building quality

Table ID: BiB_Geographic.bib_geog_house_built_env_indoor_qual

github | csv

Entity type Variables Entities Updated Related tables
Property 39 31186 2023-03-07 NA

Describes the immediate built environment of a participant’s address and indoor qualities of a participant’s home. Built environment (be) variables measure the geometry of the building and block containing the participants address. Indoor qualities (iq) variables uses Energy Performance Certificate (EPC) data to measure the properties energy use/efficiency and to describe the internal layout and features of the property.
Usage instructions: To link to BiB participants, use the property_id variable to link to bib_geog_person.
Further information: Created by Kimon Krenz, on behalf of BiB. Please reference Krenz et al. (2023) when using this data, where more detailed methodological information can be found regarding the construction of these variables.
